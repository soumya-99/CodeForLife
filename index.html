Languages For You

1. The C Trilogy

C is the mother of all programming languages. It's designed by Dennis Ritchie. It's a genaral purpose Middle Level Language cum Procedural Programming Language. It supports Structured Programming, Lexical Variable Scope, Recursion and a very powerful feature Pointer.

This is the first OOP Language in the world. C++ is a powerful general-purpose programming language. This can be used to develop operating systems, softwares, games, and so on. C++ supports different ways of programming like procedural, object-oriented, functional, and so on. This makes C++ powerful as well as flexible.

This very language is for Gamers. We can develop high end games by using this language. We can say that, this is a powerful and pro version of C language. This is also an OOP and we can do many functional work by using it.

2. Web Development

HTML stands for HyperText Markup Language. It is the is the standard markup language for documents designed to be displayed in a web browser. By this language, we can create highly customized Websites. It has tags, classes, ids and many more.

CSS stands for Cascading Style Sheets. CSS is a language that describes the style of an HTML document. CSS describes how HTML elements should be displayed. By it's help, you can create highly customize design for your website.

JavaScript is an object orient programming language designed to make web development easier and more attractive. In most cases, JavaScript is used to create responsive, interactive elements for web pages, enhancing the user experience.

Bootstrap is a free and open-source CSS framework directed at responsive, mobile-first front-end web development. It contains CSS and JavaScript based design templates for typography, forms, buttons, navigation, and other interface components.

3. Other Important Languages

Java is a general-purpose programming language that is class-based, object-oriented, and designed to have as few implementation dependencies as possible. It is used to develop desktop and mobile applications, big data processing, embedded systems, and so on. According to Oracle, the company that owns Java, Java runs on 3 billion devices worldwide, which makes Java one of the most popular programming languages.

Python is an interpreted, high-level, general-purpose programming language. Created by Guido van Rossum and first released in 1991, Python's design philosophy emphasizes code readability with its notable use of significant whitespace. Nowadays, Python is used to train machines by Machine Learning. That is called Artificial Intelligence[AI].

Kotlin is a Java Framework language and the language is friend of Android because, we can simply make android apps by this language. This language is developed by Google in 2011.
